# AI Adoption Stage Classifier

https://ai-adoption-predictor.streamlit.app/ 

**Group 3 · MBDS 2026 · ML2 Group Assignment · IE University**

Interactive Streamlit app that predicts a company's AI adoption stage (none / pilot / partial / full)
using five trained classifiers: Decision Tree, Random Forest, XGBoost, KNN, and Naive Bayes.

---

## Installation

```bash
pip install -r requirements.txt
```

## Running the app

```bash
streamlit run app.py
```

The app will open at `http://localhost:8501`.

---

## Required files

All `.pkl` files and `df_perm.csv` must be in the **same directory as `app.py`**.
They are generated by running the Jupyter notebook end-to-end.

| File | Description | How to generate |
|------|-------------|-----------------|
| `final_xgb.pkl` | Trained XGBoost classifier (best model) | `joblib.dump(final_xgb, 'final_xgb.pkl')` |
| `final_rf.pkl` | Trained Random Forest classifier | `joblib.dump(final_rf, 'final_rf.pkl')` |
| `tree_cv.pkl` | Decision Tree GridSearchCV object | `joblib.dump(tree_cv, 'tree_cv.pkl')` |
| `cvRF.pkl` | RF GridSearchCV object | `joblib.dump(cv_rf, 'cvRF.pkl')` |
| `cv_xgb.pkl` | XGBoost GridSearchCV object | `joblib.dump(cv_xgb, 'cv_xgb.pkl')` |
| `preprocessor.pkl` | Fitted sklearn ColumnTransformer | `joblib.dump(preprocessor, 'preprocessor.pkl')` |
| `df_perm.csv` | Permutation importance DataFrame | `df_perm.to_csv('df_perm.csv', index=False)` |

> **Important:** `preprocessor.pkl` must be saved **after fitting on the training set only**,
> before calling `.transform()` on either split. The DataFrame must have columns:
> `feature`, `importance_mean`, `importance_std`.

## App structure

| Tab | Content |
|-----|---------|
| Overview | Project summary, class distribution donut, pipeline visual, key metrics |
| EDA | Class imbalance, leakage detection, VIF analysis, correlation matrix, 3D scatter |
| Models | Comparison table, per-model deep dives, permutation importance, ROC/AUC, CV stability, McNemar test |
| Live Predictor | Real-time XGBoost prediction with confidence chart, SHAP explanation, next-quarter simulation |
| Findings | Technical findings, threshold guide, manager recommendations |
| PE / VC Alpha | Live inference PE screener, Leaders vs Laggards operational alpha |
| Quant Finance | Markov transitions, cost-sensitive loss, calibration, Markowitz portfolio, VaR/CVaR, survival analysis, BMA |
